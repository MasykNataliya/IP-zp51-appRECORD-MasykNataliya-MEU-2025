Основна інформація

Приклади селекторів тегів:



  body {
  font-family: "Rethink Sans", "Noto Sans", sans-serif;
  margin: 0;
  padding: 0;
  background: #fdfefc;
  color: #2a2a2a;
}

Даний селектор відноситься до тіла сторінки.

font-family: "Rethink Sans", "Noto Sans", sans-serif; - значить, що до всіх текстових елементів тіла буде застосовано шрифт за наступним алгоритмом:

  • Спочатку використовується "Rethink Sans", якщо він доступний.
  • Якщо ні — "Noto Sans".
  • Якщо жоден із них не знайдений, браузер підставить будь‑який стандартний sans-serif.

margin: 0; - прибирає всі зовнішні відступи у тега body

padding: 0; - прибирає всі внутрішні відступи у тега body

background: #fdfefc; - задає колір фону сторінки світло-зеленим.

color: #2a2a2a; - встановлює базовий колір тексту на сторінці темно‑сірим.



 header {
  background: #e8f5e9;
  padding: 28px;
  text-align: center;
  border-bottom: 2px solid #c8e6c9;
}

Даний селектор задає шапку сайту.

background: #e8f5e9; - задає світло‑зелений відтінок фону для блоку header.

padding: 28px;; - додає внутрішні відступи по 28 пікселів від всіх сторін всередині блоку header(зверху, знизу, зліва, справа).

text-align: center; - вирівнює текст та внутрішні елементи по центру горизонталі у блоці header

border-bottom: 2px solid #c8e6c9; - додає нижню рамку товщиною 2 пікселі світло-зеленого кольору



  footer {
  text-align: center;
  padding: 24px;
  background: #e8f5e9;
  border-top: 1px solid #c8e6c9;
  font-size: 15px;
  color: #2a2a2a;
}

Даний селектор задає нижню частину сторінки.

text-align: center; - вирівнює текст та внутрішні елементи по центру горизонталі у блоці footer

padding: 24px; - додає внутрішні відступи по 24 пікселів від всіх сторін всередині блоку footer(зверху, знизу, зліва, справа).

background: #e8f5e9; - Задає світло‑зелений пастельний фон у блоці footer

border-top: 1px solid #c8e6c9; - додає верхню рамку товщиною 1 піксель світло-зеленого кольору

font-size: 15px; - встановлює розмір шрифту для тексту у блоці footer

color: #2a2a2a; - задає темно‑сірий колір тексту

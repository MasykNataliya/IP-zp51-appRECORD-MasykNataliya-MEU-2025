Основна інформація

Приклади селекторів класу:



 .images {
  display: flex;
  justify-content: center;
  gap: 32px;
  margin: 40px 20px;
}

.images img {
  width: 300px;
  height: 200px;
  object-fit: cover;
  border-radius: 16px;
  box-shadow: 0 6px 12px rgba(0,0,0,0.1);
  border: 2px solid #c8e6c9;
}

Селектор класу .images

display: flex; - робить контейнер гнучким, щоб вирівнювати та розташовувати зображення в ряд

justify-content: center; - вирівнює всі елементи всередині контейнера по центру горизонталі

gap: 32px; - встановлює відстань між зображеннями 32 пікселі

margin: 40px 20px; - додає зовнішні відступи навколо контейнера: 40 пікселів зверху та знизу та 20 пікселів зліва та справа,

Селектор .images img

width: 300px; - задає ширину кожного зображення — 300 пікселів

height: 200px; - задає висоту кожного зображення — 200 пікселів

object-fit: cover; - масштабує зображення так, щоб воно заповнювало задані розміри

border-radius: 16px; - закруглює кути зображення на 16 пікселів.

box-shadow: 0 6px 12px rgba(0,0,0,0.1); - додає тінь під зображенням. Тінь має зміщення вниз на 6px з розмиттям 12px і напівпрозорий чорний колір

border: 2px solid #c8e6c9; - додає рамку світло-зеленого кольору товщиною 2 пікселі навколо зображення. 



.delete-btn {
  padding: 6px 12px;
  background: #ef9a9a;
  color: #2a2a2a;
  border: none;
  border-radius: 6px;
  cursor: pointer;
  transition: background 0.3s;
}

.delete-btn:hover {
  background: #e57373;
}

Селектор .delete-btn задає стиль кнопки для видалення даних.

padding: 6px 12px; - внутрішні відступи: 6px зверху/знизу та 12px зліва/справа

background: #ef9a9a; - задає світло‑червоний фон кнопки

color: #2a2a2a; - задає темно-сірий колір тексту

border: none; - прибирає рамку кнопки

border-radius: 6px; - закруглює кути кнопки на 6 пікселів

cursor: pointer; - при наведенні курсор змінюється на «руку», так як це клікабельний елемент

transition: background 0.3s; - додає плавний перехід кольору фону при зміні стану (наприклад, при наведенні)

Селектор .delete-btn:hover стиль елемента при наведенні курсора

background: #e57373; - при наведенні фон стає більш насиченим червоним

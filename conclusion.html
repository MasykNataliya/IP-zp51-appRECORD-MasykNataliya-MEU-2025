Висновки по роботі №2

У ході виконання лабораторної роботи №2 я закріпила практичні навички роботи з каскадними таблицями стилів (CSS). У власному WEB‑застосунку були застосовані селектори тегів, класів та ідентифікаторів, а також інші селектори.

Виконано стильове оформлення текстових елементів із використанням шрифтів, кольорів, фону, контурів, властивостей для таблиць та багаторівневих списків. Додатково застосовано просунуті CSS‑прийоми.

Таким чином, мета роботи досягнута: я отримала практичні навички застосування селекторів, властивостей CSS та стильового оформлення елементів HTML‑документів.

Контрольні питання

Для чого використовуються каскадні таблиці стилів?

CSS застосовуються для стильового оформлення HTML‑елементів: шрифти, кольори, відступи, фон, розташування. Вони відокремлюють структуру (HTML) від дизайну (CSS), що робить код чистішим і легшим у підтримці.

Які методи визначення стилів HTML‑документі ви знаєте?

  • Inline — через атрибут style у самому елементі.
    Приклад: <p style="color:blue;">Текст</p>
  • Internal — у блоці <style> всередині <head>.
  • External — окремий файл .css, підключений через <link>.

У чому різниця між принципами каскадування і спадкування?

Каскадування — коли кілька правил підходять до одного елемента, браузер вирішує, яке застосувати, враховуючи пріоритетність (inline > internal > external, специфічність селектора, порядок оголошення).
Спадкування — деякі властивості (наприклад, color, font-family) автоматично передаються від батьківського елемента до дочірніх.

Як в HTML‑документі оформлюються селектори тегу і клас?

Селектор тегу застосовується до всіх елементів певного типу:
p { color: green; }
Селектор класу застосовується до елементів із атрибутом class:
.highlight { background: yellow; }

Для чого використовується ідентифікатор?

Ідентифікатор (id) використовується для унікального елемента на сторінці. Він дозволяє застосувати специфічні стилі або взаємодіяти з елементом через JavaScript.
Приклад: #mainTitle { font-size: 40px; color: #388e3c; }

Приклади оформлення текстових елементів

З проектного сайту:

#mainTitle {
  font-size: 40px;         
  color: #388e3c;          
  text-transform: uppercase; 
  letter-spacing: 2px;       
}

form label {
  font-size: 15px;
  color: #2a2a2a;
}

.hints {
  font-size: 15px;
  line-height: 1.6;
}
